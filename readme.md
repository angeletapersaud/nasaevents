# Mod 1-Website Building: NASA Events

## Problem statement:
The purpose of this project is to provide information to users who are interested in information from NASA. One feature of this project displays an image and image information released daily by NASA. Another feature of this project utilizes google maps to display the natural events worldwide from the EONET as they occur. This website also provides the real-time location of the International Space Station as it travels around the world. 

### API's used in this project:
* NASA EONET (Earth Observatory Natural Event Tracker)
* NASA APOD (Astronomy Picture of the Day)
* NASA International Space Station Tracker 
* Google Maps
* Google Fonts


## Target End User:
Target end-user: The target end user for this project are people who have an interest in anything related to NASA.
